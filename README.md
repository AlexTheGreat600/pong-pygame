| KEY            | ABOUT              |
| -------------- | ------------------
| (=)            | increase game speed
| (-)            | decrease game speed
| (s)            | reset game speed
| (r)            | reset game scores
| (e)            | switch to easy mode
| (h)            | switch to hard mode
| (pgdown)       | move player2 up
| (pgup)         | move player2 up
| (p)            | player vs computer
| (up)           | move player up
| (down)         | move player down
| (2)            | player1 vs player2
| (c)            | computer vs computer
